# 🎁 Amigo Secreto

Um projeto simples e divertido de **Amigo Secreto**, proposto pela equipe da **Oracle Next Education** para praticar habilidades com **JavaScript e frontend puro**.

---

## ✨ Sobre o Projeto

Este projeto simula um sorteio de **amigo secreto**, onde cada participante é atribuído de forma aleatória (e secreta!) a outro.

Feito com HTML, CSS e JavaScript puro — sem frameworks ou bibliotecas externas. Uma forma prática e divertida de reforçar conceitos fundamentais da web. 🧠💡

---

## 🚀 Funcionalidades

- Adição de nomes de participantes
- Sorteio aleatório de amigos secretos
- Exibição individual e secreta dos resultados
- Interface leve e responsiva

---

## 🛠️ Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript (ES6)
